# Logic Gate Website

This website has all of the information on logic gates any beginner could need.
It includes detailed explanations as well as interactable logic gates that can be toggled by the reader.

## Features

- Interactive logic gates (AND, OR, NOT, NAND, NOR, XOR, XNOR)
- Truth tables
